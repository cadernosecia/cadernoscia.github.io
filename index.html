Política de Privacidade

Esta Política de Privacidade descreve como as informações são coletadas, usadas e protegidas.

Este aplicativo utiliza a API do Pinterest exclusivamente para criar publicações automáticas (Pins) na conta do usuário autorizada.

Nenhuma informação pessoal é armazenada, vendida ou compartilhada com terceiros. Os dados são utilizados apenas para fins de automação de postagens no Pinterest, conforme autorizado pelo próprio usuário.
